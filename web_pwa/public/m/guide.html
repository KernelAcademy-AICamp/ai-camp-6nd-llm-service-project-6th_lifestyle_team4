Daily Script 사용법

매일 한 장의 명대사로 하루를 시작하세요. 주요 기능을 아래에서 빠르게 익힐 수 있어요. 북마크·댓글·하이라이트·오늘의 한 줄은 로그인 후 사용할 수 있습니다.

refresh 오늘의 명대사 · 새로고침 누구나

홈에서 매일 한 장의 명대사 카드를 만나요. 다른 명대사를 보고 싶다면 카드 위쪽의 refresh 버튼을 누르세요.

  1. 홈 화면 오늘의 명대사 옆 refresh 버튼을 탭하면 새로운 카드가 나와요.
  2. 비회원은 하루 3번까지, 로그인하면 무제한으로 받아볼 수 있어요.
  3. 방금 본 카드는 아래 지난 기록에 쌓여 다시 펼쳐볼 수 있어요.
  4. 설정에서 맞춤 추천을 켜면 북마크와 비슷한 카드를 우선 보여줘요. (북마크 10개 이상부터 적용)
menu_book 전문 읽기 누구나

한 줄 명대사뿐 아니라 그 장면의 대본·지문 전체를 읽을 수 있어요.

  1. 카드를 탭하거나 Read Full Script 버튼을 누르면 상세 화면이 열려요.
  2. 장면 설명(SCENE), 대본·지문 전문, 작품의 의의, 작가·발표연도·조회수가 함께 나와요.
  3. 위로 쓸어내리거나 arrow_back 로 닫아요.
bookmark 북마크 · 내 서재 회원 전용

마음에 든 명대사를 보관해 라이브러리(내 서재)에 모아두세요.

  1. 카드 또는 상세 화면의 bookmark 아이콘, 혹은 Collect Script Artifact 버튼을 눌러 보관해요.
  2. 보관한 명대사는 menu_book LIBRARY 탭에서 작품별 책으로 꽂혀 모여요.
  3. 장르 칩으로 거르거나 작품 제목으로 검색할 수 있어요.
forum 댓글 · 좋아요 · 답글 회원 전용

상세 화면 맨 아래 READER NOTES에서 명대사에 대한 생각을 나눠요. (댓글 읽기는 누구나 가능)

  1. 상세 화면을 끝까지 내려 입력창에 생각을 적고 남기기를 눌러요.
  2. 다른 사람 댓글에 favorite 좋아요로 공감하거나, Reply로 답글을 달 수 있어요.
  3. 내가 쓴 댓글은 Edit·Delete로 고치거나 지울 수 있어요.
  4. 마이페이지 → 내 댓글에서 내가 남긴 글을 모아볼 수 있어요.
auto_awesome 하이라이트 회원 전용

대본 본문에서 마음에 남은 한 구절을 골라 카드로 간직하세요.

  1. 상세 화면 본문에서 원하는 구절을 길게 눌러(모바일) 또는 드래그(PC)해 선택해요.
  2. 선택하면 나타나는 + HL 버튼을 눌러요.
  3. 작품 정보와 함께 미리 보고 SAVE로 저장해요.
  4. 저장한 구절은 피드 → 하이라이트와 마이페이지 → 내 피드 → HIGHLIGHT에서 볼 수 있어요.
rate_review 오늘의 한 줄 회원 전용

북마크한 명대사에 한 줄 감상을 남겨 피드에 공유해요. (피드 읽기는 누구나 가능)

  1. dynamic_feed FEED 탭에서 우측 아래 add 버튼을 눌러요.
  2. 북마크한 명대사 중 하나를 골라요.
  3. 한 줄 감상을 적고 등록 하기를 누르면 피드 → 오늘의 한줄에 올라가요.
  4. 내가 쓴 한 줄은 마이페이지 → 내 피드 → ONE LINERS에서 고치거나 지울 수 있어요.
lock_open

북마크·댓글·하이라이트·오늘의 한 줄은 로그인 후 사용할 수 있어요. 홈 우측 상단 또는 설정에서 로그인·회원가입할 수 있고, 익명으로 담아둔 기록은 가입 시 새 계정으로 자동 옮겨집니다.
