관리자 대시보드 — Gutenberg 카테고리 피커 (전체 섹션)

Gutenberg 의 9개 섹션 전체 + 하위 카테고리를 그룹화해서 표시. 상단 검색창으로 카테고리 이름 실시간 필터링. 카테고리 클릭 → 작품 목록 → 작품 클릭 → 작품명 + 책 ID 자동 채움 → Gutenberg 본문 가져오기 + 추출 시작.

📚 Gutenberg 카테고리에서 작품 찾기 신규
· 0개 작품 (중복 제거)
행 클릭 → 작품명에 자동 입력
📖 Project Gutenberg 에서 본문 불러오기

카테고리에서 작품을 선택했으면 책 ID 가 채워져 검색 없이 바로 본문 다운로드 + 추출 시작. 직접 작품명을 타이핑한 경우는 gutendex.com 카탈로그 검색.

이 페이지는 UX 미리보기용 정적 mockup입니다. 카테고리 트리는 https://www.gutenberg.org/ebooks/categories 기반. 실제 작품 데이터는 서버 프록시 (또는 정적 dump) 연결 시 채워집니다.
