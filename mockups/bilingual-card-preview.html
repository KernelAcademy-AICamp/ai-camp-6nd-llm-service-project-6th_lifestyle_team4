관리자 · 편집

한국어/영문 동시 편집 — 한 쪽 수정 → 반대편 동기화

좌(한국어) · 우(영문) 같은 칸에 같은 필드. 본문 길이·줄바꿈 등 구조가 양쪽 모두 똑같이 보이도록 정렬. 각 필드 우측 상단 ↻ 재번역 버튼: 그 칸 한국어 → 영문(또는 영문 → 한국어) 자동 동기화.

카드 #142 편집 · 프랑켄슈타인 · 메리 셸리
🇰🇷 한국어 cards.* / works.*
🇬🇧 ENGLISH (원본) *_original
💡 빈 칸으로 저장하면 그 필드는 토글 시 한국어 그대로 표시 (영문 토글 자동 숨김)
↻ KO (필드별)

영문 칸 우측 ↻ KO 클릭 → 그 영문을 한국어로 재번역해 좌측 textarea 갱신. 다른 필드 영향 없음. 영어가 원본이므로 KO→EN 버튼은 없음.

↻ 전체 재번역 (EN → KO)

헤더 우측 버튼 → 5필드(제목·부제·작가·명대사·발췌) 영문 기준으로 한국어를 한 번에 재번역. 큰 수정 후 편의.

저장 시

양쪽 textarea 값 모두 UPDATE. 한국어 = 기본 컬럼, 영문 = *_original. 해설(설명·의의)은 단일 컬럼 그대로.

데이터 모델 — 컬럼 추가

migrations/021_bilingual.sql
-- 1) cards 본문 원본 보존
ALTER TABLE cards
  ADD COLUMN IF NOT EXISTS quote_original               text,
  ADD COLUMN IF NOT EXISTS script_excerpt_original      text,
  ADD COLUMN IF NOT EXISTS excerpt_description_original text,
  ADD COLUMN IF NOT EXISTS significance_original        text;

-- 2) works 메타데이터 원본 보존
ALTER TABLE works
  ADD COLUMN IF NOT EXISTS title_original    text,
  ADD COLUMN IF NOT EXISTS subtitle_original text,
  ADD COLUMN IF NOT EXISTS author_original   text;

-- "원본 있음" 빠른 필터/조회용 표현식 인덱스
CREATE INDEX IF NOT EXISTS cards_has_original_idx
  ON cards ((quote_original IS NOT NULL));
CREATE INDEX IF NOT EXISTS works_has_original_idx
  ON works ((title_original IS NOT NULL));

• 토글 노출 조건: works.title_original IS NOT NULL (작품 단위)
• 기존 저장 카드는 영향 없음 — *_original NULL → 버튼 자동 숨김
• 한국어 원본(번역 불필요) 작품은 *_original 비워둠 → 버튼 안 보임

동작 흐름 (확정)

  1. 1.관리자가 영문 PDF/TXT 업로드 → 카드 추출 (영문)
  2. 2."전체 번역" 또는 카드별 번역 → 각 카드에 한국어 번역 부착
  3. 3.저장 시 한국어 = 기본 컬럼, 영문 = *_original 동시 INSERT
  4. 4.PWA 홈 오늘의 한줄 — 명대사가 한국어로 표시 + 우상단 EN 토글
  5. 5.PWA 아카이브 / 관리자 목록 — 카드마다 우상단에 작은 EN pill (영문 원본 있는 카드만)
  6. 6.카드 클릭 → 상세에서 명대사 + 지문 + 설명 + 의의 모두 한국어. 우상단 EN 토글
  7. 7.EN 토글 누르면 그 카드의 모든 보이는 필드가 영문으로 즉시 스왑 (DB 추가 호출 없음)
  8. 8.목록에서 EN으로 토글 후 상세 진입 → 상세도 EN으로 유지 (sessionStorage로 카드 단위 상태 보존)
